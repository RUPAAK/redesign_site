DATABASE BACKUP SERVER

  • NodeJs/Typescript
  • Mongodb
  • Aws
  • Shell

Database backup server is a solution for backing up mongodb database and storing the backup to aws. It is made on Node Js and it's whose ultimate goal is to provide secure and easy way of backing up database. This server stores the backup to aws as well save it's link at user's local database with backup process logs. It provide live logs of backup process which can be used to show it on frontend using socket.
